## 音

### 促音

- 中間有包含 つ、ツ 的詞，要頓一下

  |        |      |
  | :----: | :--: |
  | きつぷ | 車票 |
  | ざつし | 雜誌 |
  | につき | 日記 |
  | コツプ | 杯子 |
  | ベツド |  床  |

### 長音

- 包含 AA、II、UU、EE、OO、OU、EI 等，要拉長音，片假名用 `ー` 表示

  |            |        |
  | :--------: | :----: |
  | おかあさん |  媽媽  |
  | おにいさん |  哥哥  |
  |   ゆうき   |  勇氣  |
  |  ひこうき  |  飛機  |
  |  おとうと  |  弟弟  |
  |  こうこう  |  高中  |
  |   とけい   |  手錶  |
  |  せんせん  |  老師  |
  |  スーパー  |  超市  |
  |  タクシー  | 計程車 |
