## 單字

### 促音

中間有包含 つ、ツ 的詞，要頓一下

|        |      |     |
| :----: | :--: | :-: |
| きつぷ | 車票 |     |
| ざつし | 雜誌 |     |
| につき |      |     |
| コツプ |      |     |
| ベツド |      |     |
